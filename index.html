Crypto price check

Fattah Accountancy

The Fattah Accountancy Ltd website is a responsive web application designed to provide key information about the company and its services. Built using HTML and CSS, it features a clean, professional layout that ensures optimal viewing across various devices. Key sections include "About Us," detailing the company's registration in England and Wales, and "Contact Us," which provides easy access to their email and phone number. This project effectively showcases web design principles, emphasizing usability and accessibility while establishing a strong online presence for the firm.

Weather App

This weather app is built using HTML, CSS, and JavaScript to deliver real-time weather information for any city. By utilizing the OpenWeatherMap public API, the app fetches and displays essential data such as temperature, humidity, and wind speed. Users can search for their desired location, and the app dynamically updates the weather details. Featuring a responsive design, it works seamlessly on various screen sizes, providing intuitive visuals and icons for a user-friendly experience when checking current weather conditions.

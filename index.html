Drum Kit

This is a basic drum kit web app created using HTML, CSS, and vanilla JavaScript, allowing users to play drum sounds by either clicking on drum icons or pressing specific keys on their keyboard. Each drum sound is mapped to a unique key, offering an interactive audio experience. The JavaScript code listens for both click and keypress events, triggering sound files associated with each drum. The app showcases fundamental web development skills, including DOM manipulation, event handling, and audio integration.

Live Demo GitHub
